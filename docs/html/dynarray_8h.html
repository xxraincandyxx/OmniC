Detailed Description

A generic, type-safe, stb-style dynamic array (dynarray) implementation.

This implementation uses macros and the "fat pointer" trick to provide a type-safe and convenient API that feels like a language extension.

USAGE: int* my_da = NULL; oc_da_push(my_da, 10); oc_da_push(my_da, 20); for (size_t i = 0; i < oc_da_len(my_da); ++i) { printf("%d\n", my_da[i]); } oc_da_free(my_da);
