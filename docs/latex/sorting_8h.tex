\doxysubsection{Detailed Description}
Internal sorting algorithms for the OmniC library. 

Implements various sorting algorithms on a sequential list structure. Based on standard data structure textbook algorithms (Programs 10.\+1\texorpdfstring{$\sim$}{\string~}10.7).
